# Test if the system is "unix"

Test if the operating system is "unix"

## Usage

``` r
is_unix()
```

## Value

TRUE or FALSE

## See also

Other operating system testing:
[`is_osx()`](https://ardata-fr.github.io/locatexec/dev/reference/is_osx.md),
[`is_windows()`](https://ardata-fr.github.io/locatexec/dev/reference/is_windows.md)

## Examples

``` r
is_unix()
#> [1] TRUE
```
