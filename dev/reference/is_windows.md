# Test if the system is "Windows"

Test if the operating system is "Windows"

## Usage

``` r
is_windows()
```

## Value

TRUE or FALSE

## See also

Other operating system testing:
[`is_osx()`](https://ardata-fr.github.io/locatexec/dev/reference/is_osx.md),
[`is_unix()`](https://ardata-fr.github.io/locatexec/dev/reference/is_unix.md)

## Examples

``` r
is_windows()
#> [1] FALSE
```
