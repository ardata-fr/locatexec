# Test if the system is "macOS"

Test if the operating system is "macOS"

## Usage

``` r
is_osx()
```

## Value

TRUE or FALSE

## See also

Other operating system testing:
[`is_unix()`](https://ardata-fr.github.io/locatexec/dev/reference/is_unix.md),
[`is_windows()`](https://ardata-fr.github.io/locatexec/dev/reference/is_windows.md)

## Examples

``` r
is_osx()
#> [1] FALSE
```
